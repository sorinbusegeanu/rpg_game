# Game Design Document (GDD): [Title TBD]

## 1. Overview
**Genre:** Dark Fantasy RPG
**Platform:** PC
**Core Loop:** 3D Exploration $\rightarrow$ Transition to 2D Battle $\rightarrow$ Loot & Progression $\rightarrow$ Repeat.

## 2. Setting & Aesthetics
*   **Theme:** Dark Fantasy (grim, atmospheric).
*   **Visual Style:**
    *   **Exploration:** 3D environment (Forest).
    *   **Combat:** 2D layout with "card" style presentation.
    *   **Transition:** A camera zoom and fade effect when the player enters a combat encounter.

## 3. Gameplay Mechanics
### 3.1 Combat System
*   **Type:** Turn-based / Card-based elements.
*   **Actions:** Attack, Defend, Special Ability, Item Use.
*   **Status Effects:** Poisoned, Weakened, Empowered.

### 3.2 Progression
*   **Experience Points (XP):** Gained from defeated enemies and completed quests.
*   **Leveling Up:** Increases base stats (Health, Mana, Defense).
*   **Loot:** Randomized gear drops from chests and enemy drops.

## 4. World Building
*   **Primary Region:** The Whispering Woods - A dense forest where the veil between worlds is thin.
*   **Atmosphere:** Muted colors, heavy fog, and eerie ambient sounds.

## 5. Technical Requirements
*   **Engine:** Unity / Unreal Engine (to be determined).
*   **Assets:** Hand-painted 2D assets for UI/Combat; low-poly 3D models for exploration.
